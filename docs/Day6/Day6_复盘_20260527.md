## 3. 今天掌握的知识点

- `st.set_page_config()` 用于设置浏览器标签页标题和图标。
- `st.title()` 用于设置页面主标题。
- `st.write()` 可以展示普通说明文字。
- `st.markdown()` 可以展示更结构化的 Markdown 内容。
- 项目展示不只是“代码能跑”，还要让别人快速看懂项目用途、核心能力和亮点。
- RAG Demo 的展示重点应该围绕检索、回答、来源引用和降低幻觉风险。
